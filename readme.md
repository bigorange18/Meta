# 元宇宙

​	是个人项目，该项目目标是通制作一款通过手势操纵英雄带怪升级。目前处于研发阶段，已经实现了手势拖动方块、人脸检测，后续会新增ui界面、各种英雄、野怪。

代码下载：

```bash
git clone https://github.com/gitbug1949/virtual.git
```

配置环境和安装依赖：

```bash
conda create -n py39 python=3.9.15
conda activate py39
pip install -r requirement.txt
```
